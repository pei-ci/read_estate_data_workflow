# real_estate_data_workflow
專案程式放置於code資料夾中   
結果存於result資料夾中  

使用步驟  
--
1.安裝相關套件與函式庫
```
!pip install -r requirements.txt
```
2.執行爬蟲
```
cd code
python real_estate_scraper.py
```
3.執行資料處理
```
python real_estate_data_workflow.py
```
